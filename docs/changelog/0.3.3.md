# v0.3.3 — UI Polish & Achievement Unlock Fix

## EN

### UI Improvements

- **Dashboard Bento Grid** — replaced equal-width 2-column layout with a 4-column CSS Grid; stat cards span half-width, Currently Playing spans full width; responsive (4 → 2 → 1 col)
- **Skeleton Loading** — Games page and GameDetail page now show skeleton placeholders instead of a spinner/text while data loads
- **Branded Empty States** — all empty states (no games, library empty, no filter matches, no running game) replaced with icon + message + CTA cards instead of plain alerts
- **Game Card Hover** — cards scale to 1.02× with a depth shadow on hover (150 ms ease-out)
- **Achievement Icon Size** — enlarged from 32 × 32 to 40 × 40 px in the GameDetail achievement table
- **Global % Column** — replaced raw percentage text with a mini progress bar + caption
- **Nav Active Indicator** — active nav link now shows a 3 px left border + subtle background tint
- **Page Transition** — 150 ms fade + 6 px upward slide on route change; disabled when `prefers-reduced-motion` is set
- **Contrast Fix** — secondary text bumped from `rgba(255,255,255,0.45)` to `0.55` for AA compliance
- Deleted unused Blazor template stubs: `Home.razor`, `Counter.razor`, `Weather.razor`

### Bug Fix

- **Removed random unlock jitter** — the 1–100 s random delay before each achievement unlock has been removed; achievements now unlock as soon as their scheduled offset time is reached

## 中文

### UI 改进

- **Dashboard 格栅布局** — 将等宽双列改为 4 列 CSS Grid；统计卡片各占半宽，「正在运行」卡片占满整行；响应式（4 → 2 → 1 列）
- **骨架屏加载** — Games 页和 GameDetail 页加载时显示骨架占位，替换原来的进度条/文字
- **品牌化空状态** — 所有空状态（无游戏、库为空、筛选无结果、无运行中游戏）改为带图标 + 文案 + 操作按钮的卡片，替换原纯文字提示
- **游戏卡片悬停效果** — 悬停时卡片放大至 1.02× 并显示阴影（150 ms ease-out）
- **成就图标尺寸** — GameDetail 成就表格中图标从 32 × 32 放大至 40 × 40 px
- **全球解锁率列** — 将纯文字百分比替换为 mini 进度条 + 文字标注
- **导航选中指示器** — 当前页导航项显示 3 px 左边框 + 淡色背景
- **页面切换过渡** — 路由切换时 150 ms 淡入 + 6 px 上移动画；尊重 `prefers-reduced-motion`
- **对比度修复** — 次要文字从 `rgba(255,255,255,0.45)` 提升至 `0.55`，满足 AA 标准
- 删除未使用的 Blazor 模板文件：`Home.razor`、`Counter.razor`、`Weather.razor`

### 修复

- **移除随机解锁延时** — 删除成就解锁前 1–100 秒的随机等待；成就现在在计划时间点到达后立即解锁
